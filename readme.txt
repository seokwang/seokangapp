//how to add jquery in ionic
//reference : https://forum.ionicframework.com/t/jquery-in-ionic-and-typescript/54015/2
1. npm install jquery --save

2. Install (typings and) the typings file
npm install -g typings
typings install dt~jquery --global --save

3. and finally, import the jquery to your ts file
import * as $ from ‘jquery’
